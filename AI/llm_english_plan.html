每日学习计划（25分钟）

1）背单词 10分钟

每天10个词，只记「英文→中文技术含义」，不用拼写、不用读音

2）读英文文档 10分钟

材料：GitHub README → Hugging Face 文档 → 技术博客；不会的词立刻查，读懂大意即可

3）看代码注释 5分钟

打开PyTorch/LLM项目，看英文注释→想中文含义，强化技术英语记忆

必备工具流

沉浸式翻译（Chrome插件）

看英文文档时，左边英文、右边中文，无缝对照

欧路词典/有道词典

鼠标取词，即指即译，快速查技术词汇

推荐阅读材料

1. Hugging Face Docs | 2. PyTorch Docs | 3. GitHub awesome-llm

10天小目标

  • 看懂大模型文档标题、摘要
  • 看懂GitHub项目介绍
  • 看懂训练/推理代码注释
  • 遇到生词能自己查明白

达到以上目标，即可开始学习大模型核心内容！
